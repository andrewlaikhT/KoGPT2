# KoGPT2-FineTuning
SKT-AI에서 약 20GB의 한국어 데이터를 Pre-Training 시킨 [KoGPT2](https://github.com/SKT-AI/KoGPT2)를 사용했다. 첫 번째로 가사 작사를 위해서, 정제된 한국어 가사 데이터 62MB를 Fine-tuning 한 결과물이다. 다양한 한국어 가사를 학습한 결과를 확인 할 수 있다. 

## Fine Tuning
```
python main.py
```

## generator
```
python generator.py
```

### setting
``` python
temperature = 0.7
top_p = 0.8
top_k = 40
```
